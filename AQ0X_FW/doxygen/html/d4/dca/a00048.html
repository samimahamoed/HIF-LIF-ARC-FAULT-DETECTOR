int main ( void  )

System startup function.

Configures operating system and basic system independent services

Configure Oscillator to operate the device at 60MIPS Fpll_out= Fin*M/(N1*N2)=Fin*(PLLDIV+2)/((PLLPRE+2)+(PLLPOST+1)), Fcy=Fosc/2 Fvco = Fin*M/N1 = Fin*(PLLDIV+2)/(PLLPRE+2) Fpll_out = 7.37M*65/(2*3) = 80Mhz for 7.37MHz FRC input clock

Init watchdog. Initial kick.
